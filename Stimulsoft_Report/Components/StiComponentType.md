## StiComponentType Enum

**Namespace:** `Stimulsoft.Report.Components`

### Members

| Member | Value | Description |
| --- | --- | --- |
| **Simple** |  | Simple components - only one copy in one print cycle is output. |
| **Master** |  | Master components – more than one copy in one print cycle are output. |
| **Detail** |  | Detailed components – these components are output with Master components. |
| **Static** |  | Static components – these components are the same as simple components but are output on pages only. |
