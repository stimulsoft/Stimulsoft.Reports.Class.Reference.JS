## StiImageFormat Enum

**Namespace:** `Stimulsoft.Report.Export`

Enumeration for setting format of the exported images.

### Members

| Member | Value | Description |
| --- | --- | --- |
| **Color** | 1 | Images are exported in the color mode. |
| **Grayscale** | 2 | Images are exported in the grayscale mode. |
| **Monochrome** | 3 | Images are exported in the monochrome mode. |
