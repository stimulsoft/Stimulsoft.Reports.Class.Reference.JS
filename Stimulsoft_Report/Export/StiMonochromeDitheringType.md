## StiMonochromeDitheringType Enum

**Namespace:** `Stimulsoft.Report.Export`

Enumeration describes a type of dithering for monochrome PCX file.

### Members

| Member | Value | Description |
| --- | --- | --- |
| **None** | 1 | Without dithering. Low quality, small size of file. |
| **FloydSteinberg** | 2 | Floyd-Steinberg dithering. Good quality, big size of file. |
| **Ordered** | 3 | Ordered dithering with Bayer matrix 4x4. Poor quality, medium size of file. |
